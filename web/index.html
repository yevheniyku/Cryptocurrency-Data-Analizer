    ¿Qué problema resolvemos?

    Queremos facilitar el trabajo a los posibles inversores analizando los volúmenes de compra/venta y precios de las criptomonedas en modo streaming.

    • BTC
    • BTC
    • BTC

    Datos utilizados

    Bittrex es una casa de cambios (Exchange), que permite tanto invertir en criptomonedas, como seguir la evolución del mercado en vivo.

    ¿Qué hemos creado?

    Hemos creado un script en Python que hace las consultas a la API pública por HTTPS con el método GET.
    Todas las respuestas contienen el código de respuesta, que indica si la consulta ha tenido éxito y, en tal caso, un objeto JSON con los datos consultados.

    Herramientas utilizadas

    • Bittrex API

      Nuestro script recopilará información de la API de Bittrex.

    • Apache Spark

      Análisis de datos con módulo streaming.

    • GitHub

      Plataforma de GitHub para manejar el código de una manera mas ordenada.

    • Amazon Web Services

      Proyecto alojado en un cluster de Amazon Web Services con GNU/LINUX como sistema operativo.
